Introduction to Markdown!

Markdown is a lightweight markup language with plain-text formatting syntax. Its key design goal is to be easy to read and write.

Basic Syntax

  • Headers: Use `#` for H1, `##` for H2, etc.
  • Bold: Wrap text in `**double asterisks**` or `__double underscores__`.
  • Italic: Wrap text in `*single asterisk*` or `_single underscore_`.
  • Lists: Use `-` for unordered lists and `1.` for ordered lists.
  • Links: Use `[text](URL)` for links.
  • Images: Use `![alt text](image URL)` for images.

Example

  # This is a Heading
    **This text is bold**
    *This text is italic*
